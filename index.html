02 How it works

One photo. One clear path to done.

Makeable stays with you from the first messy idea to a project you can test and share.

  1. 1

    Describe

    Say what you want to make like you’d tell a friend.

  2. 2

    Scan parts

    Put the electronics on your desk into one photo.

  3. 3

    Build + code

    Connect one piece at a time, then load working code.

  4. 4

    Test

    Check the hardware, then prove it works in the real world.

  5. 5

    Publish

    Package the guide, code, parts, and results into one project.
